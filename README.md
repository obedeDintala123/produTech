# ProduTech

ProduTech is a modern web dashboard designed to manage products, workflows, and data efficiently.  
This project focuses on performance, scalability, and a clean user experience, using modern frontend technologies.

---

## Overview

ProduTech Dashboard is a frontend-focused application built with **React, TypeScript, and Vite**, simulating backend operations using **JSON Server**.  
It demonstrates advanced UI patterns, state management, form validation, drag-and-drop interactions, and rich text editing.

---

## Features

- Modern dashboard UI
- Drag and drop functionality
- Rich text editor (TipTap)
- Form validation with Zod
- Client-side state management
- Data fetching and caching
- Mock backend with JSON Server
- Responsive and accessible UI components
- Dark / light theme support

---

## Getting Started

### 1. Clone the repository
```bash
   git clone <REPOSITORY_URL>
   cd produ-tech
```

---

### 2. Install dependencies
```bash
npm install
```

### 3. Start the development environment
```bash
npm run dev
```
